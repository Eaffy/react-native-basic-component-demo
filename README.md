# react-native-basic-component-demo
## Get Started

1. Clone the project

```
git clone --recursive https://github.com/Eaffy/react-native-basic-component-demo.git
```

2. Install node_modules

```
cd react-native-basic-component-demo && npm install
```
3. Open app

  iOS
```
npm run ios
```
  Android
```
npm run android
```
